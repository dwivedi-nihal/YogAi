Y0G A! is an AI-powered yoga instructor web application that detects yoga poses in real time and evaluates posture accuracy using computer vision and deep learning. The system guides users just like a real yoga trainer.

---

## Project Overview

- Real-time yoga pose detection using webcam
- Posture accuracy validation
- AI-based feedback for correct and incorrect poses
- User login, registration, and profile management
- Web-based and easy to use

---

## Objective

Yoga improves physical, mental, and spiritual health, but correct posture is essential.  
Y0G A! combines ancient yoga practices with Artificial Intelligence to help users practice yoga correctly without a physical instructor.

---

## Features

- Real-time human pose detection
- Yoga posture accuracy evaluation
- Correct / Incorrect pose feedback
- User authentication system
- Interactive yoga practice interface

---

## Technology Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Java, Python  
- Machine Learning: TensorFlow, PoseNet  
- Database: MySQL  
- Camera Input: RGB Webcam  

---

## Working Methodology

1. PoseNet extracts 17 human body keypoints from live video input.
2. CNN extracts spatial features such as joint angles and distances.
3. LSTM analyzes posture changes across frames for better accuracy.
4. The trained model predicts yoga poses and validates correctness.

---
